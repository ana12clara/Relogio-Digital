## **Relógio Digital** ⏰

Este repositório contém um projeto simples de Relógio Digital, desenvolvido com o objetivo de praticar e aprimorar conhecimentos em desenvolvimento web.

O projeto faz parte de uma série de pequenos projetos pessoais, criados para treinar lógica, estrutura de código e boas práticas, evoluindo gradualmente a cada novo desafio.

📚 *Sobre o Projeto*

O Relógio Digital exibe a hora atual em tempo real, atualizando automaticamente os valores de horas, minutos e segundos.

Este projeto foi desenvolvido com foco em:

Aprendizado prático

Simplicidade

Evolução contínua
